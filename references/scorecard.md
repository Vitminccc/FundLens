# 郑希框架 · 双层评分卡

给定**任意一只基金**，用双层体系评分：

- **Part A — 郑希风格契合度（6 维，100 分）**：衡量这只基金有多符合郑希的选股/选基偏好，**AI 逐维打分**（含定性判断）
- **Part B — 综合质量评分（7 维，100 分）**：衡量这只基金本身好不好，**纯量化自动计算**（由 `score_fund.py` 输出）

重要定位：Part A 衡量的是"**像不像郑希会买的基金**"，**不是**"这只基金好不好/该不该买"的绝对判断。一只基金可能很优秀但完全不符合郑希风格（如纯债、红利低波），那它在 Part A 就该低分——这是正常的，不代表基金差。务必把这层意思讲清楚，避免误导。研究与学习辅助，非投资建议。

打分流程：先跑 `python scripts/score_fund.py <代码或名称>`，它会自动：
1. 解析代码 → 备齐数据 → **计算 Part B 全部 7 维分数** → 输出完整证据档案
2. 同时输出**市场模式** + **4种买点信号** + **行业归一化**
3. 输出 Part A 所需的证据档案 → 你按下面 Part A 的标准逐维打分
4. 合成双层评分报告

# Part A — 郑希风格契合度（6 维，AI 打分）

## 六个维度（满分 100）

### 1. 景气方向 / 通胀属性（25 分）
**看什么**：前十大重仓集中在"正在涨价、景气向上"的方向吗？最好是**新技术落地、供给端创造需求**的科技型通胀（郑希最偏爱），其次是走出周期底部、走向通胀的方向。
**method 依据**：他说"我投资的最底层是研究清楚通胀从何而来……偏科技成长的通胀……是供给端创造的……爆发力往往会更强，这也是我更偏爱的方向"（method.md §1）。
**量化辅助线索**（脚本已提供）：
- 重仓股名称 → 你据此判断行业板块和景气属性
- 集中度 → 集中在少数景气方向 vs 分散在多个方向
**打分**：重仓清一色押在明确景气/通胀方向（如 AI 算力链、光通信、半导体、电力、新能源储能、有色）→ 20–25；部分沾边 → 10–19；以防御/红利/纯债/低景气方向为主 → 0–9。

### 2. ROE 低位弹性（20 分）
**看什么**：持仓偏好是不是"ROE 从低到高的修复弹性"——供求压制型（行业被压制、公司能力不止于此）或预先研发型（研发投入大、方向对路、即将兑现）？还是已经是高 ROE 白马？
**method 依据**："我喜欢 ROE 低的资产……投资景气方向上标的资产 ROE 从低到高的过程。当下越低，未来向上的空间就会大"（method.md §4b）。
**量化辅助线索**：
- 脚本会提供规模与资产配置 → 规模偏小 + 高股票仓位 → 更可能持有中小市值
- 个股 ROE 本 skill 不取，标"需核实"
**打分**：明显偏好低 ROE 待修复/中小成长 → 15–20；混合 → 8–14；清一色高 ROE 成熟白马 → 0–7。

### 3. 全球视野 / 中国比较优势（15 分）
**看什么**：方向是否落在"全球技术周期 + 中国有比较优势"的环节（他选光通信的逻辑）。
**method 依据**：method.md §3。
**打分**：方向明确踩在中国全球占优的环节（光通信、部分半导体设备/材料、电力设备、锂电）→ 11–15；一般 → 5–10；与全球技术周期无关的纯内需/主题 → 0–4。

### 4. 流动性（10 分）
**看什么**：重仓股是否有足够流动性、能进退；基金规模与持仓风格是否匹配（规模过大 + 偏小盘会损流动性）。
**method 依据**："第一是个股的流动性……即使选股有所失误，也能及时撤出"（method.md §4a、§5）。
**量化辅助线索**：
- 脚本提供基金规模和资产配置 → 大规模 + 小盘持仓 = 流动性风险
**打分**：重仓多为流动性好的标的、规模适配 → 7–10；一般 → 4–6；大量小票 + 大规模、流动性堪忧 → 0–3。

### 5. 集中度与周期拼接（15 分）
**看什么**：是否**适度集中**于当期景气环节，并且**动态调仓**（高换手、周期拼接）而非买入躺平？脚本会给"前十大集中度"和"季度间换手代理"。
**method 依据**："复利来源视为周期的一次次拼接……高换手有时候是必要的"（method.md §6）。
**打分**：集中在景气环节 + 季度间有明显调仓（换手代理高）→ 11–15；集中但几乎不动，或非常分散无主线 → 5–10；既分散又不动 → 0–4。
**注意**：换手高在这套框架里是**加分**（符合周期拼接），不是缺点。

### 6. 业绩与回撤的客观印证（15 分）
**看什么**：真实收益/任职回报/最大回撤。郑希认为"股票是进攻性资产……靠选对景气方向降回撤"，所以**高弹性高波动是其风格的自然结果**——重点不是回撤小，而是"是否靠选对景气方向赚到了景气的钱"。
**method 依据**：method.md §5 回撤观。
**量化辅助线索**（脚本已提供）：
- 区间收益率（近1年/3年/成立以来）+ 同类对比
- 最大回撤 + 同类百分位排名
**打分**：长期/任职回报显著跑赢同类与沪深300、且收益来自景气方向 → 11–15；一般 → 5–10；长期跑输或与景气无关 → 0–4。

# Part B — 综合质量评分（7 维，自动计算）

全部由 `score_fund.py` 自动计算并输出，无需 AI 介入。以下为计算规则供参考和验证。

## 七个维度（满分 100）

### 1. 收益能力（25 分）
**核心指标**：加权收益率
**计算方法**：按分析周期（默认 90d）对各时段收益率加权，映射到 0-25 分。
- ≥100% → 25 分
- 0% → 12.5 分
- ≤-50% → 0 分
- 线性插值

**分析周期加权表**：

| 分析周期 | 权重分配 |
|---|---|
| 7天 | 周60% + 月30% + 季10% |
| 30天 | 月40% + 季25% + 半年20% + 年15% |
| 90天(默认) | 季40% + 半年30% + 年20% + 月10% |
| 180天 | 半年40% + 年35% + 季15% + 2年10% |
| 365天 | 年45% + 2年25% + 半年15% + 3年15% |

### 2. 风险控制（20 分）
**核心指标**：近1年最大回撤
**计算方法**：0%回撤 = 20分，50%+回撤 = 0分，线性映射。

### 3. 收益质量（15 分）
**核心指标**：卡玛比率（年化收益 ÷ |最大回撤|）
**计算方法**：
- 卡玛 ≥ 3 → 15 分
- 卡玛 = 0 → 5 分
- 卡玛 < 0 → max(0, 5 + calmar) 分
- 无数据 → 5 分（中间值）

### 4. 基金评级（10 分）
**核心指标**：4机构评级（上海证券/招商证券/济安金信/晨星）
**计算方法**：
- 可用评级取均值（1-5 星），映射 0-7 分
- 5 星数量加成 0-3 分（每个5星+1分，上限3分）
- 无评级数据 → 5 分（中间值，不因新基金扣分）

### 5. 基金经理（10 分）
**核心指标**：经理人数 + 基金公司
**计算方法**：
- 1人管理 = 7 分（专注）
- 2人管理 = 5 分
- 3人+ = 3 分
- 头部基金公司加成 +3 分

### 6. 持仓质量（10 分）
**核心指标**：收益一致性 + 风格漂移
**计算方法**：
- 一致性(60%)：短期(90天)方向与长期(365天)方向一致 → 100%，反向 → 0%
- 漂移(40%)：|短期收益 - 长期收益| / max(|长期收益|, 0.01)，漂移 ≤ 0.2 → 100%，≥ 2.0 → 0%
- 综合 = 一致性 × 0.6 + 漂移分 × 0.4，映射到 0-10

### 7. 赚钱效应（10 分）
**核心指标**：正收益概率
**计算方法**：月/季/半年/年四期正收益占比，权重 0.1/0.2/0.3/0.4。
- 100% → 10 分
- 50% → 5 分
- 0% → 0 分

# 市场模式权重切换

根据沪深300近60日收益率自动判断：

| 模式 | 判断条件 | 收益能力 | 风险控制 | 收益质量 | 赚钱效应 |
|---|---|---|---|---|---|
| 均衡 | -10% ≤ ret ≤ +15% | 25 | 20 | 15 | 10 |
| 牛市 | ret > +15% | 35 | 10 | 15 | 10 |
| 熊市 | ret < -10% | 15 | 30 | 20 | 5 |

基金评级/基金经理/持仓质量三维度不受市场模式影响。

市场模式加权总分 = 受影响维度按新权重重算 + 不受影响维度原分。

# 4种买点信号

| 信号 | 条件 | 默认阈值 | 说明 |
|---|---|---|---|
| 回撤买点 | 近1年最大回撤 ≥ 阈值 | 10% | 回撤已释放风险，可能处于低位 |
| 评分买点 | 综合评分 ≥ 阈值 | 80 | 高质量基金（AI买点未触发时才记录，互斥） |
| 连跌买点 | 连续3交易日下跌 或 单日跌幅>2% | — | 短期超跌反弹机会 |
| AI买点 | 评分≥70 + 赚钱效应≥80%权重 + 风控<50%权重 | 超跌反弹 | 高质量+赚钱效应好+风险未失控，适合超跌买入 |

**AI买点和评分买点互斥**：AI买点优先，未触发时才记录评分买点。

# 行业归一化

评分最终按同类百分位归一化：
- 利用 `同类排名走势` 的最新 rank/peer_count 算百分位
- `percentile = (1 - rank / total) × 100`
- 归一化分数 = 百分位% × 市场模式加权总分
- 新基金无同类数据时跳过归一化，标注"(新基金，同类数据不足，未归一化)"

---

# 双层评分交互规则

当两层评分差异大时，需要桥接说明：

| 风格契合度 | 综合质量 | 桥接说明 |
|---|---|---|
| 高 | 高 | 郑希风格的优质标的 |
| 高 | 低 | 像郑希风格但当前表现不佳，需关注 |
| 低 | 高 | 优质基金，但风格与郑希不同 |
| 低 | 低 | 既不符合郑希风格，综合质量也偏弱 |
